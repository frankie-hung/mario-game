# Super Mario Game

A classic side-scrolling platformer game built with HTML5 Canvas and vanilla JavaScript. Jump, collect coins, defeat enemies, and reach the flag to win!

## ğŸ¯ How to Play

### Controls

- **Arrow Keys** or **WASD**: Move left/right
- **Space**, **Arrow Up**, or **W**: Jump
- **X**: Shoot fireballs (when you have Fire Power)

### Gameplay Tips

1. Hit question blocks (?) from below to reveal power-ups
2. Stomp enemies from above to defeat them
3. Collect coins to increase your score
4. Avoid falling off platforms or getting hit by enemies
5. Use fireballs strategically to defeat enemies from a distance
6. Star Power makes you invincible and faster - use it wisely!

## ğŸš€ Getting Started

### Prerequisites

No installation required! This game runs entirely in the browser using vanilla JavaScript.

### Play Online

**[Play the game directly in your browser](https://frankie-hung.github.io/mario-game/)** - No download required!

### Running Locally

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Start playing!

Alternatively, you can use a local web server:

```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (with http-server installed)
npx http-server

# Using PHP
php -S localhost:8000
```

Then navigate to `http://localhost:8000` in your browser.
